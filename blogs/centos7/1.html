    # cnetos7虚拟机配置静态网络

    # 设置虚拟机静态ip

    2.配置静态网络

    [root@master ~]# vi /etc/sysconfig/network-scripts/ifcfg-ens33  # 文件名一般都是 ifcfg- 加 ens33(上面查出来的)

BOOTPROTO=static  # 修改成static
ONBOOT=yes   # 修改成yes 开机自启
IPADDR=192.168.137.130  # 网段要对应上vmware 里的nat网段
NETMASK=255.255.255.0   # 这个统一使用这个
GATEWAY=192.168.137.1   # 对应wmware中nat的网关
DSN1=8.8.8.8       # 这个随意可以换其他dns
DNS2=114.114.114.114
[root@master ~]# service network restart  # 重启网卡
Restarting network (via systemctl):                        [  确定  ]

    # 设置vmware nat 的网段

    1. 在VMware的编辑->虚拟网络编辑器->更改设置 ->点击VMnet8
    2. 子网ip对应虚拟机上的网段我这里设置成 192.168.137.0
    3. 子网掩码还是 255.255.255.0
    4. 点击 NAT设置，把网关设置成 192.168.137.1 # 这里对应虚拟机上的网关
    5. 点击应用 和确定

    # 设置本机的网络

    1. 打开 控制面板\网络和 Internet\网络连接
    2. 打开VMnat8 属性
    3. 更改ipv4
    4. IP地址改为 192.168.137.2 注意：这里的不能和虚拟机网关一样的设置为 192.168.137.1 不然主机可以ping通虚拟机，虚拟机ping不通主机，并且不能上网
    5. 默认网关要设置成你当前能上网 网络的网关，不然不能联网
    6. dns随便设置一个
